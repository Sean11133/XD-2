## 10. 部署架構

本次變更為純前端 TypeScript 程式庫的 Services 層功能擴充，不涉及後端部署、資料庫遷移或 CI/CD 環境變更。

| 步驟     | 命令                                       | 說明                       |
| -------- | ------------------------------------------ | -------------------------- |
| 安裝依賴 | `cd file-management-system && npm install` | 現有依賴，應無新套件       |
| 執行測試 | `npm run test`                             | Vitest 全套測試            |
| 覆蓋率   | `npm run test -- --coverage`               | 驗證 ≥ 90% 覆蓋率          |
| 建置     | `npm run build`                            | 確認無 TypeScript 編譯錯誤 |
